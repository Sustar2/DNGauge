DNGauge Windows 便携版
======================

1. 包内文件说明
---------------

- `DNGauge.exe` : 主程序，可直接双击运行
- `DNGauge.ico` : 程序图标
- `README_RUN.txt` : 快速入口说明
- `README_RUN_CN.txt` : 中文说明
- `README_RUN_EN.txt` : English guide

2. 如何启动
-----------

1. 解压整个文件夹
2. 保持文件夹内文件在同一目录下
3. 直接双击 `DNGauge.exe`

3. 支持的输入格式
-----------------

- 普通图片：`jpg`、`jpeg`、`png`、`tif`、`tiff`、`bmp`、`webp`
- `rawpy/libraw` 支持的相机 RAW
- `dng`
- plain `.raw`

4. 主要功能
-----------

- 左右图对比
- 同步缩放和平移
- RAW / DNG / JPG / PNG 混合对比
- `Value:显示(渲染)` 与 `Value:RAW` 取样
- plain `.RAW` 手动参数输入与 RAW 调参

5. 注意事项
-----------

- 不需要单独安装 Python 环境
- 如果 SmartScreen 弹出提示，请选择“更多信息”后再决定是否运行
- 请保持整个文件夹内容完整，不要单独移动 `DNGauge.exe`
