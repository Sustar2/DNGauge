DNGauge Linux Portable
======================

Please read one of these guides:

- `README_RUN_EN.txt` : English quick guide
- `README_RUN_CN.txt` : 中文使用说明

Quick start:
1. Extract the whole folder.
2. Keep all files in the same directory.
3. Double-click `DNGauge.desktop` or `DNGauge`.

Optional:
- run `./install_desktop_launcher.sh`
- then launch `DNGauge` from the app menu or desktop shortcut
