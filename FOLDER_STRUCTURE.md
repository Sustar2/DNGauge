# DNGauge Folder Structure

## 1. Source files kept in the repo root

- `shotwell_compare.py`
  - main application entry and UI logic
- `README.md`
  - main project documentation
- `SHOTWELL_MAPPING.md`
  - Shotwell RAW pipeline notes
- `requirements.txt`
  - Python dependencies
- `run.sh`
  - local Linux run helper for the source version

## 2. Packaging folder

- `packaging/`
  - stores everything related to packaging and distribution

Contents:
- `packaging/DNGauge.spec`
  - PyInstaller config
- `packaging/DNGauge.png`
  - Linux window / launcher icon
- `packaging/DNGauge.ico`
  - Windows executable icon
- `packaging/build_linux.sh`
  - builds `dist/DNGauge`
- `packaging/build_windows.bat`
  - builds `dist/DNGauge.exe` on Windows
- `packaging/package_portable_linux.sh`
  - creates the portable Linux folder and `.tar.gz`
- `packaging/portable_assets/DNGauge.desktop.template`
  - template used to generate launchers
- `packaging/portable_assets/install_desktop_launcher.sh`
  - installs a desktop/app-menu launcher into the user's home directory
- `packaging/portable_assets/README_RUN.txt`
  - entry readme
- `packaging/portable_assets/README_RUN_CN.txt`
  - Chinese end-user guide
- `packaging/portable_assets/README_RUN_EN.txt`
  - English end-user guide

## 3. Generated folders

- `build/`
  - PyInstaller temporary build cache
  - safe to delete
- `dist/`
  - generated executable output
  - safe to delete and rebuild
- `release/`
  - generated portable package output
  - safe to delete and regenerate

## 5. Cleanup decisions made

Removed as redundant:
- root `DNGauge.desktop`
  - local absolute-path launcher was redundant with generated desktop launchers
- `packaging/portable_assets/DNGauge.sh`
  - simple wrapper around `DNGauge` was redundant because the binary is directly runnable

Kept intentionally:
- `DNGauge.desktop.template`
  - needed by `install_desktop_launcher.sh`
- both Chinese and English readmes
  - intended for distribution to colleagues

## 6. Recommended workflow

For development:
- edit source files in the repo root
- run with `python shotwell_compare.py` or `./run.sh`

For Linux packaging:
- run `./packaging/build_linux.sh`
- run `./packaging/package_portable_linux.sh`

For sharing with colleagues:
- send `release/DNGauge-linux-portable.tar.gz`
