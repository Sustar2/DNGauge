# DNG_COMPARE

一个用于 **DNG/RAW 双图对比** 的桌面工具（PyQt5），支持：
- 双图/单图显示
- 同步缩放与拖拽
- Shotwell 风格 RAW 渲染参数（见 `SHOTWELL_MAPPING.md`）
- 曝光/对比度/高光/阴影等调整
- LOCATE 取样（渲染值 / RAW 原值）

## 4. 使用说明（功能总览）

顶部按分组：

- **输入**：`L↥` / `R↥`（加载左/右图）
- **布局**：`布局·单/双`、`同步·开/关`、`←`
  - `←`：按住时右图覆盖左图，松开恢复
- **调参**：`调参·开/关`
  - 打开后右侧显示调整面板（Exposure/Contrast/Saturation/Temperature/Tint/Highlights/Shadows）
- **LOCATE**：`Locate·开/关`、`Value:显示(渲染)` / `Value:RAW`
  - 点击图像后，底部显示 `LOCATE INFO`
- **视图**：`100%`、`Fit`

底部信息栏固定按 **左图 -> 右图** 顺序显示取样值（双图模式）。

## 5. 常见问题

## Q1: 看起来“糊”
- 先点 `100%` 看 1:1 细节；
- `Fit` 只是适配窗口，不代表原像素显示。

## Q2: 取样值是 RAW 吗？
- `Value:显示(渲染)`：显示渲染后的 RGB/Gray；
- `Value:RAW`：显示 RAW CFA 单通道原始值（R/G/B 之一）。

## Q3: 双图不同步？
- 检查按钮 `同步·开/关` 是否为开。
